# Context Mapping

Context maps describe relationships between bounded contexts.

## Relationship Types

- Partnership
- Shared Kernel
- Customer/Supplier
- Conformist
- Anti-Corruption Layer
- Open Host Service
- Published Language
- Separate Ways

## Checks

- Which context is upstream?
- Which context is downstream?
- Who owns the contract?
- Which team controls change?
- Where is translation needed?
- Where does a shared model create coupling?
- Where should contexts stay separate?

## Output Pattern

| Upstream | Downstream | Relationship | Contract | Risk |
