# Domain-Driven Design

Use this skill for deep DDD work.

## Modes

- `/ddd discover`
- `/ddd context-map`
- `/ddd subdomains`
- `/ddd aggregate`
- `/ddd events`
- `/ddd event-storm`
- `/ddd review`
- `/ddd diagram`

## Diagrams

Use `/ddd diagram` for context maps, subdomain maps, aggregate boundaries,
domain event flows, event storming summaries, and ubiquitous language maps in
Mermaid or PlantUML.
